# Система управления рецептами

Проект для хранения рецептов, масштабирования ингредиентов и формирования списка покупок.

## Установка

git clone https://github.com/drnvxx/HW_2_OOP_Testing_Git.git

cd HW_2_OOP_Testing_Git

pip install -r requirements.txt

## Использование

Для запуска тестов:

pytest
