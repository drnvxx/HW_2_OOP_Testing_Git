# Система управления рецептами

Простое приложение на Python, которое позволяет создавать рецепты блюд, масштабировать количество инградиентов в зависимости от порций и формировать итоговый список покупок.

## Установка

Чтобы развернуть проект, необходимо выполнить следующие команды в терминале:
1) Клонировать репозиторий к себе на компьютер:
git clone https://github.com/drnvxx/HW_2_OOP_Testing_Git.git
2) Перейти в папку проекта:
cd HW_2_OOP_Testing_Git
3) Установить необходимые для работы зависимости:
pip install -r requirements.txt

## Использование

Можно импортировать разработанные классы в свои скрипты

Для запуска тестов выполнить в терминале команду:
pytest
